# leyusportsm-archive

本仓库用于归档和发布多个独立 HTML 页面。所有页面均为静态文件，不依赖后端服务，可直接通过浏览器打开浏览。

## 项目简介

这是一个通用型 HTML 页面归档仓库，旨在整理和保存一批独立制作的网页素材。页面内容涵盖不同主题，但均以 HTML 格式呈现，便于本地查看或部署到任何静态托管平台。

本仓库不针对任何特定域名或网站，仅作为文件存储与版本管理的载体。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式表、脚本等）
├── README.md       # 本文件
└── LICENSE         # 仓库许可证
```

- `pages/` 目录下每个 HTML 文件即为一个独立页面，文件名通常反映其内容或用途。
- `assets/` 目录用于存放页面所需的辅助资源，按需组织子目录。

## 页面归档说明

- 所有页面均以标准 HTML5 编写，不依赖外部 API 或数据库。
- 页面之间相互独立，无跨文件依赖（公共资源除外）。
- 归档文件保持原始结构，不进行额外修改或优化。
- 如需引用或分发，请保留文件完整性及本说明。
